# IVF AI 患者助手 - 产品需求文档 (PRD)
## Product: FertilityBuddy (暂定名)

**版本**: MVP v1.0  
**日期**: 2025年11月  
**目标**: 6周内完成可测试的MVP

---

## 1. 产品概述

### 1.1 产品定位
**一句话描述**: IVF患者的24/7 AI伴侣，提供日程管理、用药指导、情感支持和知识科普。

### 1.2 目标用户
**主要用户**:
- 正在进行IVF治疗的女性（25-45岁）
- 焦虑度高、需要情感支持
- 对医学术语不熟悉
- 希望更好地理解治疗过程

**次要用户**:
- 伴侣（共享进度）
- 准备开始IVF的女性（观望期）

### 1.3 核心价值主张
- **减少焦虑**: 24/7随时回答问题，不用等医生
- **不会遗漏**: 自动提醒用药、检查
- **更懂流程**: 用简单语言解释医学术语
- **不孤单**: 情感支持，理解她们的痛苦

### 1.4 成功指标（MVP阶段）
- 10个测试用户使用超过2周
- 平均每天互动3次以上
- NPS分数 > 8
- 至少5个愿意付费

## 2. 核心功能（MVP范围）

### ✅ Must Have（MVP必须有）

#### 2.1 AI对话助手
**功能描述**:
- 用户可以随时问问题
- AI用简单、温暖的语言回答
- 支持中英文

**使用场景**:
```
用户: "我今天感觉很恶心，这正常吗？"
AI: "打促排针后感到轻微恶心是常见的副作用，很多姐妹都会经历。通常2-3天会好转。

如果你：
- 恶心到无法进食
- 伴随剧烈腹痛
- 持续超过3天

请立刻联系你的医生。

现在你能吃得下东西吗？需要我给你一些缓解恶心的小tips吗？"
```

**技术要求**:
- 使用Claude API (claude-sonnet-4)
- 对话历史保存（至少最近50条）
- 响应时间 < 3秒

**限制**:
- ⚠️ 不能给医疗建议（"我不是医生，这只是科普信息"）
- ⚠️ 紧急情况立刻建议联系医生
- ⚠️ 识别危机信号（自杀倾向、严重抑郁）→ 提供心理热线


**API调用示例**:
```typescript
// app/api/chat/route.ts
import Anthropic from '@anthropic-ai/sdk';

export async function POST(req: Request) {
  const { message, userId } = await req.json();
  
  // 获取用户信息
  const user = await getUserById(userId);
  
  // 获取最近10条对话历史
  const history = await getConversationHistory(userId, 10);
  
  const anthropic = new Anthropic({
    apiKey: process.env.ANTHROPIC_API_KEY,
  });
  
  const response = await anthropic.messages.create({
    model: 'claude-sonnet-4-20250514',
    max_tokens: 1000,
    system: SYSTEM_PROMPT
      .replace('{userName}', user.name)
      .replace('{ivfStage}', user.ivf_stage)
      .replace('{city}', user.city),
    messages: [
      ...history, // 历史对话
      { role: 'user', content: message }
    ],
  });
  
  // 保存对话
  await saveConversation(userId, message, 'user');
  await saveConversation(userId, response.content[0].text, 'assistant');
  
  return Response.json({ 
    reply: response.content[0].text 
  });
}
```

## 10. 风险和挑战

### 10.1 技术风险
- ⚠️ Claude API成本（需要控制对话长度）
- ⚠️ 推送通知可靠性（PWA vs Native）
- ⚠️ 数据隐私（HIPAA合规？）

**缓解措施**:
- 限制对话历史长度（最近50条）
- 提供email提醒备份
- 只存储必要数据，不存储PHI

### 10.2 产品风险
- ⚠️ 用户可能过度依赖AI
- ⚠️ AI可能说错话导致恐慌
- ⚠️ 处理失败和悲伤很困难

**缓解措施**:
- 明确disclaimer（"我不是医生"）
- 严格prompt engineering
- 危机情况立刻引导找专业人士

### 10.3 市场风险
- ⚠️ 用户可能不愿意付费
- ⚠️ 竞争对手出现
- ⚠️ 获客困难（敏感话题）

**缓解措施**:
- MVP免费，证明价值后再收费
- 快速迭代，保持优势
- 通过IVF社群和论坛获客
